# ADS - Exercícios de Programação I (PROG1)

Repositório criado para armazenar e documentar a minha evolução prática na disciplina de Programação I, como parte do curso superior de Tecnologia em Análise e Desenvolvimento de Sistemas (ADS).

## 🚀 Tecnologias Utilizadas
* Python 3

## 📁 Organização do Repositório
À medida que as aulas avançarem, os exercícios serão organizados nas seguintes estruturas:
* `01-conceitos-basicos/`: Variáveis, entradas, saídas e operadores aritméticos.
* `02-estruturas-condicionais/`: Tomadas de decisão (`if`, `elif`, `else`).
* `03-estruturas-repeticao/`: Laços de repetição (`while`, `for`).
* `04-funcoes-e-listas/`: Modularização e estruturas de dados.

## 📈 Meu Objetivo
Consolidar a lógica de programação, aprender boas práticas de versionamento com Git/GitHub e construir uma base sólida para os próximos semestres.
